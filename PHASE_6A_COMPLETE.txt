📊 KEY METRICS TRACKED
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. P&L (Profit & Loss)
   • Realized P&L:     Closed positions (actual profit)
   • Unrealized P&L:   Open positions (paper profit)
   • Total P&L:        Realized + Unrealized

2. Win Rate
   • Wins:             Number of profitable trades
   • Losses:           Number of losing trades
   • Win Rate %:       (Wins / Total) × 100

3. Risk-Adjusted Returns
   • Sharpe Ratio:     Return / Volatility
     > 2.0 = Excellent
     1.0-2.0 = Good
     0-1.0 = Acceptable
     < 0 = Poor

4. Risk Metrics
   • Max Drawdown:     Largest loss from peak
   • Profit Factor:    Total Wins / Total Losses
   • ROI:              (Total P&L / Invested) × 100

5. Trade Analysis
   • Average Win:      Mean profit of winning trades
   • Average Loss:     Mean loss of losing trades
   • Largest Win:      Best single trade
   • Largest Loss:     Worst single trade

🔄 HOW IT WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

TRADE LIFECYCLE:

1. Order Submitted
   ├─ Status: pending
   ├─ OrderHash: stored
   └─ Waiting for fill...

2. Polymarket Webhook: ORDER_FILLED
   ├─ Update status: filled
   ├─ Store entry price
   ├─ Create position (status: open)
   └─ Start tracking unrealized P&L

3. Position Open
   ├─ Current price updates
   ├─ Calculate unrealized P&L
   └─ Show in dashboard

4. Position Closed
   ├─ Store exit price
   ├─ Calculate realized P&L
   ├─ Update position (status: closed)
   └─ Update agent stats

5. Performance Calculated
   ├─ Win rate updated
   ├─ Sharpe ratio calculated
   ├─ ROI computed
   └─ Leaderboard updated
