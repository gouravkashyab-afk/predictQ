🔒 SAFETY ARCHITECTURE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

7 LAYERS OF PROTECTION:

1. Default State:       simulateOnly: true (all agents start safe)
2. Explicit Opt-In:     User must enable with warning
3. Agent Permission:    Agent must be active and authorized
4. Spending Limits:     Per-trade ($50) and daily ($200) caps
5. Balance Check:       Verifies sufficient USDC before trading
6. Error Handling:      Automatic fallback to simulation on errors
7. Emergency Stop:      Panic button to pause all agents
